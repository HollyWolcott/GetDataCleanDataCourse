Construction of this dataset is described in "README.md". The dataset was constructed using the `run_analyis.R` script. The tidy dataset may be found in "tidy.txt" and the variables are described here in the file "codebook.md".

Each row in "tidy.txt" corresponds to one subject and activity. The following variables are included:

* subject - Each ID corresponds to one subject in the dataset.
* activity - Factor variable describing the type of activity performed by the subject. Possible values: "WALKING", "WALKING\_UPSTAIRS","WALKING\_DOWNSTAIRS","SITTING","STANDING","LAYING"
